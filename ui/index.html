Hybrid Chess

An asymmetric board game where International Chess faces Chinese Chess on a shared 9×10 board.
Full AlphaZero RL pipeline, C++ engine, and interactive web UI.

♟ Play Now 📊 Analyze Replays
🎮

Human vs AI

Play as either side against built-in AI agents — Random, Greedy, or AlphaBeta search.

🧠

RL Training

Train your own AlphaZero agent with a single command. CPU & GPU supported.

⚡

C++ Engine

High-performance move generation and alpha-beta search via pybind11.

⚖️

Balance Variants

Rule variants to explore asymmetric game balance — no queen, extra cannon, and more.

⚡ Quick Start

# Install & play
pip install -r requirements.txt
python -m hybrid.server

# Train your own AI
python -m scripts.train_az_iter --iterations 20 --games 100
